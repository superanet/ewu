# 公司落地页技术栈分析

## 项目需求
- 小型公司官网（日活<1000）
- SEO优化 + 云服务器部署

---

## 技术栈

### 1. 前端框架

**Next.js 15 + React 19 + TypeScript**
- 原生SSR/SSG支持，SEO友好
- 全栈框架，内置API Routes
- TypeScript类型安全

### 2. 样式方案

**Tailwind CSS 4 + Shadcn/ui**
- 原子化CSS，开发效率高
- Shadcn/ui提供高质量可定制组件

---

### 3. 数据库与ORM

**SQLite（开发环境）+ PostgreSQL（生产环境）**
- 开发环境：SQLite零配置，快速启动
- 生产环境：PostgreSQL生产级，并发性能好

**Drizzle ORM**
- 轻量高性能
- TypeScript原生支持
- 同时支持SQLite和PostgreSQL

---

### 4. 部署方案

**Node.js 20 LTS + PM2**
- PM2进程管理，自动重启和负载均衡

**Nginx**
- 反向代理和静态文件服务
- SSL/TLS终止（Let's Encrypt免费证书）

### 5. 开发工具

**包管理器**：pnpm

**代码质量**：ESLint + Prettier + TypeScript

**富文本编辑器**：Tiptap

**图片处理**：Sharp + Next.js Image组件

### 6. 支付方案

**微信支付 + 支付宝**
- 官方SDK：wechatpay-node-v3（微信）+ alipay-sdk（支付宝）
- 支持Native扫码支付、H5支付、JSAPI支付
- 异步回调处理和订单状态管理
